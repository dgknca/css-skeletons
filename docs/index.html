css-skeletons

A single <div>, dynamic, fully customizable CSS library.

Custom Variables

Custom Variable Description Default Valid Class
--c-w (container-width) Width of the container. 300px All
--c-p (container-padding) Padding of the container. 10px
--c-bg (container-background) Background of the container. transparent
--bg Background of the shapes. #e2e1eb
--shine Color of the shine animation. rgba(255,255,255, 0.2)
--t Duration of the shine animation. 2s
--lines Count of the lines.

.skeleton-line .skeleton-rect .skeleton-circle-multi-line .skeleton-card-2 .skeleton-list

--g-gap (group-gap) Gap between the groups. E.g. circle and other lines. 15px

Helper Classes

Class Description Required Class
.adapt-to-circle No --lines needed. Just need to set --c-s (circle-size). The lines will be produced and placed automatically based on the size of the circle.
.skeleton-circle-multi-line
.no-animate Cancels the animation.

Examples

.skeleton-line 
<div class="skeleton skeleton-line"></div>
--l-h (line-height)

If you want to change the default values, just send it in style="". 

<div class="skeleton skeleton-line" style="--l-h:40px;"></div>
--lines

No height needed. The height will be calculated automatically based on the number of lines and other default values. 

<div class="skeleton skeleton-line" style="--lines:3"></div>
--l-gap (lines-gap) 
<div class="skeleton skeleton-line" style="--lines:3;--l-gap:20px;"></div>
--c-bg (container-background) 
<div class="skeleton skeleton-line" style="--lines:3;--c-bg: #F8F7FC;"></div>
--bg (shape-background) 
<div class="skeleton skeleton-line" style="--lines:3;--c-bg:#faecf6;--bg:#eeb6e0;"></div>
--shine (shine-color) 
<div class="skeleton skeleton-line" style="--lines:3;--c-bg:#faecf6;--bg:#eeb6e0;--shine:#f5c3da;"></div>

Are custom properties too long? You can store them in a class. 

<style>
.my-class {
    --lines: 3;
    --c-bg: #faecf6;
    --bg: #eeb6e0;
    --shine: #f5c3da;
}
</style>

<div class="skeleton skeleton-line my-class"></div>
--c-p (container-padding) 

<div class="skeleton skeleton-line" style="--lines:3;--c-bg: #F8F7FC;--c-p: 30px;"></div>
.skeleton-rect
--rect-h (rectangle-height)

We can add another big rectangle with rect class. 

<div class="skeleton skeleton-rect" style="--rect-h:150px;--lines:3;"></div>
--g-gap (group-gap)

Gap between groups. e.g. rect and other lines or circle and other lines. 

<div class="skeleton skeleton-rect" style="--lines:3;--rect-h:150px;--g-gap:30px;"></div>
.skeleton-circle
--c-s (circle-size. Default 40px)

Simple circle.

<div class="skeleton skeleton-circle" style="--c-s: 60px;"></div>
.skeleton-circle-line

Side by side circle and line.

<div class="skeleton skeleton-circle-line"></div>
--c-s (circle-size) 
<div class="skeleton skeleton-circle-line" style="--c-s:60px"></div>
.skeleton-circle-multi-line

If you want to use multiple lines, you should use "circle-multi-line" class. 

<div class="skeleton skeleton-circle-multi-line" style="--c-s:80px;--lines:4;"></div>
.adapt-to-circle

Just set --c-s (circle-size). The lines will be produced and placed automatically based on the size of the circle. 

<div class="skeleton skeleton-circle-multi-line adapt-to-circle" style="--c-s:150px;"></div>
.skeleton-chart-columns
--cols (Count of columns. Default 5, max 10)
--col-w (column width. Default 25px)
--col-gap (Gap between columns. Default 25px)
--chart-h (chart-height. Default 200px)

You don't have to set --c-w (container-width). The width will be calculated automatically based on the number of columns count and size of columns gap. 

<div class="skeleton skeleton-chart-columns"></div>
<div class="skeleton skeleton-chart-columns" style="--cols: 10;--col-w:50px;--col-gap:30px;--chart-h:300px;"></div>
.skeleton-chart-line 
<div class="skeleton skeleton-chart-line" style="--c-w: 500px;"></div>
--chart-btm (height of the bottom area of ​​the chart. Default 40px) 
<div class="skeleton skeleton-chart-line" style="--c-w: 400px;--chart-h: 150px;--chart-btm:60px;"></div>
.skeleton-card-1
--card-h (card-height. Default 150px)
--g-gap is 30px as default for .skeleton-card-1
--c-s is 30px as default for .skeleton-card-1

Dimensions of the lines and the big rectangle will be calculated automatically based on the --c-w (container-width). 

<div class="skeleton skeleton-card-1" style="--c-w: 600px;"></div>
.skeleton-card-2 
<div class="skeleton skeleton-card-2" style="--c-w: 600px;--lines:3;"></div>
.skeleton-list 
<div class="skeleton skeleton-list" style="--lines:5;"></div>
