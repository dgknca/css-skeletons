All Parameters

.skeleton {
    --t: 2s; /* animation duration */
    --bg: #f0f0f0; /* background of shapes */
    --c-bg: transparent; /* container-background */
    --shine: #ecebeb; /* shine animation color */
    --c-p: 10px; /* container-padding */
    --c-w: 300px; /* container-width */
    --lines: 1; /* count of lines */
    --l-h: 20px; /* line-height */
    --l-gap: 10px; /* gap between lines */
    --g-gap: 15px; /* gap between groups. like circle and other lines */
    --c-s: 40px; /* circle-size */
    --rect-h: 80px; /* rectangle-height */
}

Line

.line 
<div class="skeleton line"></div>
--l-h (line-height)

If you want to change the default values, just send it in style="". 

<div class="skeleton line" style="--l-h:40px;"></div>
--lines

No height needed. The height will be calculated automatically based on the number of lines and other default values.

<div class="skeleton line" style="--lines:3"></div>
--l-gap (lines-gap) 
<div class="skeleton line" style="--lines:3;--l-gap:20px;"></div>
--c-bg (container-background) 
<div class="skeleton line" style="--lines:3;--c-bg: #F8F7FC;"></div>
--bg (shape-background) 
<div class="skeleton line" style="--lines:3;--c-bg:#faecf6;--bg:#eeb6e0;"></div>
--shine (shine-color) 
<div class="skeleton line" style="--lines:3;--c-bg:#faecf6;--bg:#eeb6e0;--shine:#f5c3da;"></div>

Are custom properties too long? You can store them in a class.

<style>
.my-class {
    --lines: 3;
    --c-bg: #faecf6;
    --bg: #eeb6e0;
    --shine: #f5c3da;
}
</style>

<div class="skeleton line my-class"></div>
--c-p (container-padding) 

<div class="skeleton line" style="--lines:3;--c-bg: #F8F7FC;--c-p: 30px;"></div>
.rect --rect-h (rectangle-height)

We can add another big rectangle with rect class.

<div class="skeleton rect" style="--rect-h:150px;--lines:3;"></div>
--g-gap (group-gap)

Gap between groups. e.g. rect and other lines or circle and other lines.

<div class="skeleton rect" style="--lines:3;--rect-h:150px;--g-gap:30px;"></div>

Circle

.circle --c-s (circle-size. Default 40px)

Simple circle.

<div class="skeleton circle" style="--c-s: 60px;"></div>
.circle-line Default

Side by side circle and line.

<div class="skeleton circle-line"></div>
--c-s (circle-size) 
<div class="skeleton circle-line" style="--c-s:60px"></div>
.circle-multi-line

If you want to use multiple lines, you should use "circle-multi-line" class.

<div class="skeleton circle-multi-line" style="--c-s:80px;--lines:4;"></div>
.adapt-to-circle

Just set --c-s (circle-size). The lines will be produced and placed automatically based on the size of the circle.

<div class="skeleton circle-multi-line adapt-to-circle" style="--c-s:150px;"></div>

Charts

.chart-area --chart-h (chart-height. Default 200px) 
<div class="skeleton chart-area" style="--c-w: 500px;"></div>
--chart-bottom (height of the bottom area of ​​the chart. Default 40px) 
<div class="skeleton chart-area" style="--c-w: 800px;--chart-h: 300px;--chart-bottom:80px;"></div>
.chart-columns --cols (Count of columns. Default 5, max 10) --col-w (Column width. Default 25px) --col-gap (Gap between columns. Default 25px) --chart-h (Height of the chart. Default 200px)

You don't have to set --c-w (container-width). The width will be calculated automatically based on the number of columns count and size of columns gap.

<div class="skeleton chart-columns"></div>
<div class="skeleton chart-columns" style="--cols: 10;--col-w:50px;--col-gap:30px;--chart-h:300px;"></div>

Cards

.skeleton-card-1 --card-h (Height of the card. Default 150px) --g-gap (Group gap. Between big rectangle and others. Default 30px) --c-s (Circle size. Default 30px)

Sizes of the three lines and the big rectangle will be calculated automatically based on the container width (--c-w).

<div class="skeleton skeleton-card-1" style="--c-w: 600px;"></div>
<div class="skeleton skeleton-card-1" style="--c-w: 1200px;--card-h: 300px;--c-s: 50px;--l-gap:20px;--g-gap: 60px;"></div>
